User Restrict to Create Product and Partner

This module will help you to add restrictions on users to create product and partner in odoo



App Features
  • Restrict users to create product and partner
  • Compatible with community and enterprise
How to use

In product view create button is visible for users

In customer view create button is visible for users

You need to set group for restrict create product and partner

Now user cannot create product from kanban view

Also user cannot create product in form and tree view

Also user cannot create product from sale order, purchase order, invoices etc..

Restricted users cannot create customer/vendor in odoo

Also partner create restricted in sale order, purchase order, invoice, etc..
