[Original] POS Manager Validation using Employee PIN

Way to have manager validation for specific features in POS

Features:

  • Validate POS Closing
  • Validate Order Deletion
  • Validate Order Line Deletion
  • Validate Order Payment
  • Validate Discount Application
  • Validate Price Change
  • Validate Decreasing Quantity
  • Compatible with Point of Sale / Manager Validation using Employee PIN / Badge ID

Important: Manager settings uses Employee (hr.employee) table, not User (res.users) table! Login with Employees must be checked for the configuration.

Configuration

Easy setup of manager validation

Manager validation configuration is as easy as ABC. Configuration can be found in Point of Sale > Configuration > Settings if Login with Employees is enabled.

Specify managers (employees) in Manager Validation (Employee) tab for each point of sale. Then select which features in POS you want to have manager validation.

POS Interface

Manager validation seemless integration to POS interface

There will be Manager Validate window that will ask the manager's passcode/password in order to proceed with the action.

Note that the passcode can be configured in Employees > Employees > Employees > HR Settings tab.

Compatibility

This module is compatible with Odoo 16.0 Community and Enterprise Editions
